App API

GET /health

Returns the health of the API.

Request

Headers
Accept: application/json
Parameters
None

200 - Success

Headers
Content-Type: application/json
Body
{
    "type": "object",
    "properties": {
        "status": {
            "type": "string",
            "enum": ["UP", "DOWN", "UNKNOWN"]
        }
    },
    "additionalProperties": false,
    "required": ["status"]
}
